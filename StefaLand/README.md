This is the StefaLand Package for Foundation model training

# Installation

### Create a new environment, then activate it

```Shell
conda create -n Stefaland python=3.8
conda activate Stefaland
```

### 1) Using PyPI

Install our hydroDL stable package from pip (Python version>=3.0)

```
cd ./Stefaland
pip install -e .
```

### 2) Using Conda

```
cd ./Stefaland
conda develop .
```
